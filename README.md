## Getting Started
First, Create a `.env`file in root directory to store your jwt secret
```
JWT_SECRET=XXXX.....
```

then run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
